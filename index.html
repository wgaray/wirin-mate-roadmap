Technical Spotlights

120FPS WebGPU

Unified graphics pipeline batch-rendering all waveforms, meters, playhead and automation lanes through a single shared GPU device.

MIDI Precision

Native SMF integration with microsecond-accurate event scheduling, full piano-roll editor and a dedicated velocity-lane GPU renderer.

ID-Sync Telemetry

WIRN Binary Protocol v2 streams per-track gain, pan, mute, solo and automation state with zero-copy decoding under 0.1ms per frame.

Sample-Accurate Undo

Backend Memento-Command engine wires every region, mix and automation edit into a single transactional undo/redo stack — Ardour-grade reliability.
